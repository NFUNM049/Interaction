### 期末项目主题：世界各地的教育程度和入学率

数据：
* 世界各国的教育程度数据表
* 世界各国的入学率
* 世界GDP排名

我们生活的基本需求之一是教育，当一个国家大多数人口都受过教育之后，这个国家就被称为受教育的国家。一个国家的经济增长掌握在平民手中，如果他们没有受到足够的教育，就无法取得进步。

目的是利用可视化查看世界各地受教育程度情况和入学率情况，结合世界的经济情况进行下一步分析经济与教育程度的关系。
